# libclassreader #

An __experimental__ C library for reading Java .class files

## Dependencies ##

This library depends on GLib 2 mostly for its datastructures like strings,
hashes and lists. It is built using cmake 3.0 or newer.

## Build It ##

```bash
$ cmake .
$ make
$ make install
```
